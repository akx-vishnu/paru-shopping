# 🛒 Paru Shopping

A modern, image-rich e-commerce web app for **Paru Shopping** — a local grocery & daily essentials store based in **Kochi, Kerala**.

Live → [paru-shopping.vercel.app](https://paru-shopping.vercel.app)

---

## ✨ Features

- **Image-dominant UI** — Full-photo hero, category strips with food photos, deal image banners
- **WhatsApp Ordering** — Customers build a cart and send the order directly via WhatsApp (`+91 90040 63606`)
- **5 Categories** — Groceries, Fruits & Vegetables, Household Items, Personal Care, Snacks & Beverages
- **25 Products** — Five products per category with images, prices, and descriptions
- **Smart Shop Page** — Image-based category selector pinned below navbar, search, price filter, quick-add hover
- **Cart & Checkout** — Add/remove items, quantity control, multi-step checkout (Cart → Shipping → WhatsApp send)
- **Mobile First** — Sticky bottom nav bar, horizontal scroll strips for categories & deals, responsive grids
- **SEO Ready** — Meta description, Open Graph, Twitter Card, canonical URL, theme-color, `robots: index,follow`

## 🚀 Running Locally

```bash
git clone https://github.com/akx-vishnu/paru-shopping.git
cd paru-shopping
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
