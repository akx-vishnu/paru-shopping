# Paru Shopping

A modern e-commerce web application for **Paru Shopping**, based in Kochi, Kerala. 
Built to provide a smooth, fast, and responsive user experience for local shopping.

## Features
- **Responsive Design**: Mobile-first approach with a sticky bottom navigation bar for better UX on smaller screens.
- **Routing**: Client-side routing for seamless transitions between Home, Shop, Cart, Contact, and About pages.
- **Cart Management**: Context API-based cart management.
- **Tech Stack**: React 19, Vite, Tailwind CSS v4, React Router v7.

## Running Locally

1. Clone the repository
2. Install dependencies with `npm install`
3. Start the dev server with `npm run dev`
